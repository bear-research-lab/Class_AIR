This folder contains the code and resources for Week 2 of the course. Note that the codes are generated using LLMs, so there may exist some bugs. Code examples include:

- pr_code1: Simple forward kinematics for a 2-link planar robot arm.
- pr_code2: Basic coordinate transformation between two frames in 2D space.
- pr_code3: 2D Coordinate transformation in serial chain robots.
- pr_code4: Drawing a 2D humanoid using 2D Coordinate transformation.
- pr_code5: Basic coordinate transformation in 3D space.
- pr_code6: Demonstration of euler angle based rotation in 3D space. It uses explicit rotation matrices for roll, pitch, and yaw rotations.
- pr_code7: Demonstration of quaternion based rotation in 3D space. It uses implicit rotation matrices. 
- pr_code8: Demonstration of gimbal lock phenomenon issue in 3D space.
- pr_code9: Demonstration of quaternion based rotation in 3D space.
